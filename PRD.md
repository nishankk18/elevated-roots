# Elevated Roots MA — Mobile App PRD

## Overview
- **Project:** Elevated Roots MA Cannabis Dispensary Mobile App
- **Platforms:** iOS + Android (React Native)
- **Type:** Ecommerce / Pickup-focused cannabis retail
- **Locations:** Abington, Halifax, Kingston (MA)
- **Stack:** React Native + TypeScript, Zustand, TanStack Query, Firebase/OneSignal, Stripe-compatible gateway, Headless CMS

## Categories
Flower, Pre-Rolls, Vaporizers, Edibles, Extracts, Topicals, Tinctures

## User Roles
Shopper, Admin/Content Manager, Store Staff

## Navigation
Bottom tabs: Home, Shop, Cart, Rewards, Account
Global header: store selector, search, notifications

## MVP Features
1. Age Gate (21+) + Compliance Disclaimer
2. Store Selection (Abington, Halifax, Kingston)
3. Home Screen (banners, categories, featured, best sellers, reorder)
4. Product Catalog (search, sort, filter, infinite scroll, store-specific)
5. Product Detail (gallery, variants, THC/CBD, add to cart, favorites)
6. Cart (persistent, store-specific)
7. Checkout (guest + account, pickup flow, order confirmation)
8. User Account (signup/login, profile, orders, favorites)
9. Rewards / Loyalty (Roots Rewards placeholder)
10. Content Pages (About, Locations, Blog, Careers, Contact — CMS-driven)
11. Push Notifications
12. Admin-ready API architecture

## Data Models
Store, Product, User, Cart, Order, Content
